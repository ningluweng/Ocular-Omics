# GEO to RIS Converter

## HOW IS THE DATA MAPPED

- **Accession IDs** → Listed as “authors”
- **Organism** → Bracketed in the title (“[Homo sapiens]”)
- **Platform & dataset info** → Authors field (for optimal search and sort)
- **GEO** → Listed as the “journal”  
- **Description** → Included as the abstract  
- **FTP link** → Used as the DOI/URL
